This program provides an interface for users to query the iMessage database.

1) Use New Path and input the path of the location of chat.db. Then press the Enter button
2) The conversations will then be listed along with the phone number of the recipient.

Known issues
If a connection to chat.db cannot be made then make a copy of chat.db and specify the path to this one. The reason may be MacOS may locks chat.db so it cannot be accessed by another user at the same time.
